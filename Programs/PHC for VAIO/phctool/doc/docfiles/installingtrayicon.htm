installing the Tray icon

back
The PHCTool tray icon sits in your systems panel and offers you the following things:
  • open PHCTool by clicking on this icon
  • reload your values each time the Icon is loaded (normally at the same time your Desktop is loaded)

the PHCTool tray icon (in a Gnome Panel)

To add the PHCTool tray icon to Gnome autostart function open your session settings.

the Gnome Session

Click "add" to add a new program. As Name you can enter what ever you want.
The command line is the complete patch to PHCTool and the file name "pgctray.sh".
The comment is optional.

Adding a new Program to the autostarter

After clicking "ok" the new entry should be visible.

the PHCTool tray with PHCTool added

If everything is okay you'll see the icon next time you log on.
To manually call the Tray icon (for bug finding for example) you just need to call the phctray.sh from a command box.
