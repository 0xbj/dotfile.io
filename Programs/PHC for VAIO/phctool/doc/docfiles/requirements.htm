System Requierements

back
In order to run PHCtool you need the following things:
  • a CPU that is supported by linux-phc
  • linux-PHC Version 0.3.0 or higher
  • Python
  • Python bindings to Gnome
  • PyGTK
  • Python-Gnome2-Extras
  • Python bindings to Glade
  • Python bindings to GTK
  • an GTK environment (Gnome or another Desktop with Gnome files installed


Since I (the developer) am using Ubuntu you should automatically meet all requierements. Just make sure that linux-PHC is working.
